# IT Suppport Virtual Lab

This repository documents my hands-on IT support ticketing lab using Spiceworks Help Desk. The project focuses on troubleshooting common technical issues, documenting resolutions, and developing skills in ticket management, networking, and customer support.

-Solve the issue

-Document the solution
